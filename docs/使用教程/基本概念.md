# Yuml 独有的基本概念

## 1. 块与元素
在 Yuml 中，每个根节点（Root）下都包含一个或多个“块”（Block）。  
例如：
```yaml
a:
  _: 1

b:
  _: 1
  _2: 1
```

这里，a 和 b 都是块（Block），分别称为 a 块 和 b 块。
块是 Yuml 代码结构的基本单元，用于组织和封装代码内容。
每个块中的代码:前部分均为元素名和元素值, , 即a块中包含名为_的元素, 值为1, b块包含名为_和_2的元素, 值均为1
